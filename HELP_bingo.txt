Ce mode de jeu est jouable de 1 à 4 joueurs. Si vous êtes seul, choisissez une couleur (On vous conseille le vert).
Ce n'est pas synchronisé. Il faut donc une personne qui maintient le bingo, idéalement, quelqu'un qui ne joue pas et qui sert donc d'arbitre.
L'objectif de ce jeu est de marquer le plus de points en réalisant les propositions du carré. Chaque proposition vaut 1 point mais réaliser une ligne/colonne/diagonale entière vous donnera un bonus (3 points par défaut, modifiable dans les paramètres). Le jeu dure un maximum de 30 minutes (modifiable dans les paramètres). Mais les joueurs peuvent se mettre d'accord pour arrêter avant. Les cases ne sont pas exclusives (elles peuvent être réalisées par plusieurs joueurs).

- Chaque joueur se met en position
- Appuyez sur "Commencer" pour remplir les cases
- Les joueurs peuvent prendre le temps d'analyser le contenu des cases pour se faire un plan de progression
- Quand tous les joueurs sont prêts, appuyez sur "Démarrer Chrono". Les joueurs peuvent alors bouger et tenter de réaliser les propositions.
- Quand un joueur complète la proposition d'une case, appuyez sur sa couleur puis sur la case concernée. En cas d'erreur, re-cliquer sur la case enlèvera la couleur.
- Le jeu s'arrête automatiquement quand le temps est écoulé. On peut aussi l'arrêter en appuyant sur "Arrêter Chrono". Une fenêtre avec les scores s'affiche alors.
- Notez qu'il est possible à tout moment de faire une pause. Il suffit d'appuyer sur "Arrêter Chrono" puis quand vous voulez reprendre, appuyez sur "Démarrer Chrono".
